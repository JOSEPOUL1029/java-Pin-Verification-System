PIN Verification System
A simple Java console application that verifies a user's PIN with a maximum of three attempts.

Features
-	Hardcoded PIN ("1234")
-	User input using the "Scanner" class
-	Maximum of 3 attempts
-	Displays remaining attempts after incorrect entries
-	Grants access for the correct PIN
-	Denies access after 3 failed attempts
  
How to Run
-	javac PinVerificationSystem.java
-	java PinVerificationSystem

  
Example
-	Enter PIN: 1111
-	Incorrect PIN. Attempts left: 2
-	Enter PIN: 1234
-	Access Granted
Technologies Used
-	Java
-	Scanner Class
-	Loops
-	Conditional Statements
